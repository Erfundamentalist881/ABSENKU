View your app in AI Studio: https://ai.studio/apps/f9a90076-c041-420c-b51e-c654838f2c7f

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
